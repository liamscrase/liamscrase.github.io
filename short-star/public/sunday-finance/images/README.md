# Sunday Finance page assets

Export assets from the Figma file (Sunday-Pictures, node 8-56) and place them here so the page displays correctly.

Suggested filenames:

- `hero-bg.png` – Hero background (billboard image)
- `logo.png` – Header logo
- `drivers-seat.svg` – Hero “drivers seat” headline graphic
- `check.svg` – Checkmark icon for lists
- `trade-in.jpg` – Trade-in section image
- `arrow.svg` – “Car finance by people who love cars” arrow graphic
- `sunday-drive-graphic.png` – Sunday Drive graphic
- `sunday-drive-logo.png` – Sunday Drive logo
- `expert-advice.jpg` – Expert car advice section image
- `trade-placements.jpg` – Trade-in placements image
- `inspections.jpg` – Vehicle inspections image
- `icon-phone.svg` – Footer contact icon
- `icon-visit.svg` – Footer visit icon
- `icon-message.svg` – Footer get in touch icon
- `social.svg` – Social media icons (optional)

You can also use the Figma Dev Mode asset URLs while the Figma desktop app is running with the plugin (localhost:3845).
